## ADDED Requirements

### Requirement: 输入时显示目录建议

系统 MUST 在用户编辑新建会话的工作目录时自动显示目录建议。

#### Scenario: 输入非空路径
- **当** 用户在工作目录输入框中输入非空文字
- **则** 系统展开目录建议
- **并且** 在防抖延迟后搜索匹配目录
- **并且** 搜索完成后显示匹配目录

#### Scenario: 清空输入框
- **当** 用户清空工作目录输入框
- **则** 系统展开目录建议
- **并且** 显示根目录下的可选目录
- **并且** 保留最近使用路径列表（如有）

#### Scenario: 搜索进行中
- **当** 目录搜索请求尚未完成
- **则** 系统在建议列表中显示加载状态

#### Scenario: 搜索无结果
- **当** 目录搜索完成但没有匹配目录
- **则** 系统不显示目录结果
- **并且** 保留最近使用路径列表（如有）

### Requirement: 聚焦时初始化目录建议

系统 MUST 在工作目录输入框获得焦点时按当前输入内容初始化目录建议。

#### Scenario: 聚焦已有内容的输入框
- **当** 工作目录输入框获得焦点且已有内容
- **则** 系统展开目录建议
- **并且** 搜索当前输入内容对应的目录

#### Scenario: 聚焦空输入框
- **当** 工作目录输入框获得焦点且内容为空
- **则** 系统展开目录建议
- **并且** 显示根目录下的可选目录
- **并且** 显示最近使用路径列表（如有）

#### Scenario: 软键盘打开时
- **当** 工作目录输入框获得焦点并弹出软键盘
- **则** 系统调整页面可用高度
- **并且** 目录建议不被软键盘遮挡

### Requirement: 选择建议后填充工作目录

系统 MUST 在用户选择目录建议后填充工作目录并收起建议列表。

#### Scenario: 选择搜索结果目录
- **当** 用户点击建议列表中的目录结果
- **则** 系统将工作目录设置为该目录的完整路径
- **并且** 收起建议列表
- **并且** 将该路径加入最近使用路径

#### Scenario: 选择最近使用路径
- **当** 用户点击建议列表中的最近使用路径
- **则** 系统将工作目录设置为该路径
- **并且** 收起建议列表
